Create a draft
===
`bin/draft`: Adds a draft post. Drafts are apparently [controversial][1], but I like to capture things with as little resistance as possible, so it fits my workflow.

Serve with `bundle exec jekyll serve --drafts`

{% highlight bash %}
{% include draft %}
{% endhighlight %}
